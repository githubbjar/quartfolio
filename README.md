# Quartfolio

Personal portfolio site built with Laravel, Tailwind CSS, and Alpine.js.

Live: https://quartfolio.com

---

## Overview

Quartfolio is a custom-built portfolio platform designed to showcase editorial design, web development, and digital publishing work.

The site reflects my approach to building complete web solutions—from concept and design through development and deployment.

---

## Approach

Rather than relying on a traditional CMS, Quartfolio is built from scratch using Laravel. This allows for full control over data structure, routing, and rendering logic.

The goal is a system that is:
- Flexible and maintainable  
- Structured for long-term growth  
- Tailored specifically to the content it presents  

---

## Architecture

- Projects are modeled explicitly in the database  
- Slug-based routing for clean, readable URLs  
- Image handling via Laravel storage  
- Lightweight interactivity with Alpine.js  
- Component-driven frontend using Tailwind CSS  

---

## Tech Stack

- Laravel (PHP)  
- Tailwind CSS  
- Alpine.js  
- Vite  

---

## Status

Actively maintained and evolving.  
Serves as both a public portfolio and a working codebase reflecting current development practices.
